Welcome to my penetration testing blog!


On this site, I will show how to obtain privileged root user access on the various virtual machines hosted on the platform of HackTheBox at HackTheBox.com as well as write about various other pen testing topics. Each walkthrough will include screeshots of the key steps and outputs so you can easily follow along. The walkthroughs will be structured as follows in line what with the steps of a traditional pen test:

  1. Machine exploitation matrix and summary of attack path
  2. Enumeration
  3. Obtaining initial foothold
  4. Lateral movement (if applicable)
  5. Privilege escalation to root user or Administrator
  6. Lessons learned and vulnerability summary


Feel free to use the tags or categories buttons on the left sidebar to filter based on the type of asset or vulnerability you are interested in learning about. Please feel free to comment on my work and ask any questions! I will get back to you as soon as I can.
